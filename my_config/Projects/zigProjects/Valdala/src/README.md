# Modules

| module | content |
| -- | -- |
| algebra | vectors, matrices, quaternions |
| asset | statically embedded assets |
| client | client-side application loop |
| coordinate | layout of and conversion between coordinate systems |
| game | shared game logic |
| glfw-wgpu | glue code between GLFW and wgpu-native (might go into a separate library) |
| graphics | rendering of scene objects |
| gui | window handling, user interface elements, input events |
| module | loading and managing game modules and their resources |
| scene | 3d render state |
| terrain | terrain, chunk and tile state |
